# AQA GCSE PE Revision Tracker

A simple revision tracker to help students monitor their progress.

## Features
- Click topics to mark as completed
- Saves progress in browser
- Easy to edit topics in `data.json`

## How to Use
1. Upload files to GitHub
2. Go to Settings → Pages
3. Enable GitHub Pages (main branch)
4. Open the provided link

## Customisation
Edit `data.json` to add/remove topics.
